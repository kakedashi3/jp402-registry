AI がそのまま払う

x402 = HTTP 402 を返すと、エージェントが署名して即支払い。アカウント登録もカード入力も不要。エージェントが顧客になる。

🇯🇵

円建て・経理がラク

JPYC は円と 1:1 の電子決済手段。為替リスクなし、適格請求書・電子帳簿保存法（電帳法）と相性が良く、国内事業者がそのまま使える。

⚡

数行・ガスレス

既存 API に 402 応答を足すだけ。EIP-3009 で買い手はガス代不要、代金は P2P で直接入金。
