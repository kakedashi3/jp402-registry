なぜ今

「広告を見てもらう」から「エージェントに払ってもらう」へ

AI が人の代わりに探して使い、買う時代。API 提供者の収益化は 「無料公開＋広告／月額」から「呼ばれた分だけ、その場で課金」へ動きます。 それを円建て・国内法準拠で実現する組み合わせが JPYC × x402 です。

🤖

AI がそのまま払う

x402 = HTTP 402 を返すと、エージェントが署名して即支払い。アカウント登録もカード入力も不要。機械が顧客になる。

🇯🇵

円建て・経理が綺麗

JPYC は円と 1:1 の電子決済手段。為替リスクなし、適格請求書・電帳法と相性が良く、国内事業者がそのまま使える。

⚡

数行・ガスレス

EIP-3009 で買い手はガス代不要。既存 API に 402 応答を足すだけ。P2P 直接着金で資金を預からない設計。
